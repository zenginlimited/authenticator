Authenticator

Zengin Authenticator is a Chrome Extension designed to securely generate and manage authentication codes for your accounts. Developed by the Zengin team, it simplifies multi-factor authentication (MFA) while keeping your data private and secure.

Features

  • Generate TOTP (Time-based One-Time Password) codes
  • Support for multiple accounts
  • Simple and user-friendly interface
  • Secure local storage of secret keys
  • Quick copy to clipboard for authentication codes
  • Fully compatible with Chrome and Chromium-based browsers
  • Paste codes directly from Gmail
    1. Requires gmail.readonly scope to parse recent emails containing an OTP (one-time password)
    2. Optionally mail.google.com scope to discard used OTP emails

Installation

  1. Clone or download this repository: 
    git clone https://github.com/zenginlimited/authenticator.git
  2. Open Chrome and go to chrome://extensions
  3. Enable Developer mode (top-right toggle)
  4. Click Load unpacked and selected the cloned repository folder

The extension should now appear in your browser toolbar.

Usage

  1. Click the Zengin Authenticator icon in your Chrome toolbar
  2. Add an account using a QR code or secret key, or scan the page to automatically add your secret
  3. Copy the generated authentication code to the input box

The extension should now appear in your browser toolbar.
